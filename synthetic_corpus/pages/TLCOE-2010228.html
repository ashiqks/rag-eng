1. Test Plan

1.1 Problem Statement

On Athleta Checkout, the current control experience does not sufficiently support checkout start rate for the cart abandoners segment. Based on prior time savings-tactic learnings across the premium tier, we want to evaluate whether the proposed change moves the needle within the 120-day window.

1.2 Hypothesis

If we apply 'Wallet pay promotion' on Athleta Checkout (iOS Push) to cart abandoners, then we expect a lift of at least 0.34% on Checkout Start Rate because promoting wallet pay above the billing form shortens checkout for mobile users.

1.3 Experimental Changes

  • Control: existing Checkout experience on Athleta iOS Push.
  • Challenger: Wallet pay promotion variant — promoting wallet pay above the billing form shortens checkout for mobile users.
