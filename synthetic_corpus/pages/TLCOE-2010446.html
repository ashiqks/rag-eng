1. Test Plan

1.1 Problem Statement

On Banana Republic Factory PDP, the current control experience does not sufficiently support checkout completion rate for the cart abandoners segment. Based on prior time savings-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the 90-day window.

1.2 Hypothesis

If we apply 'PDP default size' on Banana Republic Factory PDP (Mobile Web) to cart abandoners, then we expect a lift of at least 1.42% on Checkout Completion Rate because defaulting to the most-purchased size for the current sku saves a tap.

1.3 Experimental Changes

  • Control: existing PDP experience on Banana Republic Factory Mobile Web.
  • Challenger: PDP default size variant — defaulting to the most-purchased size for the current sku saves a tap.
