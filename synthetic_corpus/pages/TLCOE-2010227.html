1. Test Plan

1.1 Problem Statement

On Athleta PLP, the current control experience does not sufficiently support email capture rate for the cart abandoners segment. Based on prior quality-tactic learnings across the premium tier, we want to evaluate whether the proposed change moves the needle within the 120-day window.

1.2 Hypothesis

If we apply 'PLP affinity sort' on Athleta PLP (Tablet Web) to cart abandoners, then we expect a lift of at least 1.49% on Email Capture Rate because ml-ranked sort using prior browsing affinity should outperform default 'featured' sort.

1.3 Experimental Changes

  • Control: existing PLP experience on Athleta Tablet Web.
  • Challenger: PLP affinity sort variant — ml-ranked sort using prior browsing affinity should outperform default 'featured' sort.
