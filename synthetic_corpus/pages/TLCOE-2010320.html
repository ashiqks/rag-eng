1. Test Plan

1.1 Problem Statement

On Banana Republic PDP, the current control experience does not sufficiently support checkout completion rate for the cart abandoners segment. Based on prior time savings-tactic learnings across the premium tier, we want to evaluate whether the proposed change moves the needle within the 60-day window.

1.2 Hypothesis

If we apply 'PDP single-step selection' on Banana Republic PDP (iOS Native) to cart abandoners, then we expect a lift of at least 1.06% on Checkout Completion Rate because merging size and colour selection into a single visible block reduces decision time.

1.3 Experimental Changes

  • Control: existing PDP experience on Banana Republic iOS Native.
  • Challenger: PDP single-step selection variant — merging size and colour selection into a single visible block reduces decision time.
