1. Test Plan

1.1 Problem Statement

On Athleta PDP, the current control experience does not sufficiently support upt for the high-aov customers segment. Based on prior time savings-tactic learnings across the premium tier, we want to evaluate whether the proposed change moves the needle within the 60-day window.

1.2 Hypothesis

If we apply 'PDP single-step selection' on Athleta PDP (Desktop) to high-aov customers, then we expect a lift of at least 0.74% on UPT because merging size and colour selection into a single visible block reduces decision time.

1.3 Experimental Changes

  • Control: existing PDP experience on Athleta Desktop.
  • Challenger: PDP single-step selection variant — merging size and colour selection into a single visible block reduces decision time.
