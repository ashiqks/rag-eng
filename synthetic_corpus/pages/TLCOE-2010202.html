1. Test Plan

1.1 Problem Statement

On Banana Republic PDP, the current control experience does not sufficiently support upt for the first-time buyers segment. Based on prior time savings-tactic learnings across the premium tier, we want to evaluate whether the proposed change moves the needle within the -105-day window.

1.2 Hypothesis

If we apply 'PDP default size' on Banana Republic PDP (Tablet Web) to first-time buyers, then we expect a lift of at least 1.03% on UPT because defaulting to the most-purchased size for the current sku saves a tap.

1.3 Experimental Changes

  • Control: existing PDP experience on Banana Republic Tablet Web.
  • Challenger: PDP default size variant — defaulting to the most-purchased size for the current sku saves a tap.
