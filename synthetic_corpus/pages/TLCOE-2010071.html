1. Test Plan

1.1 Problem Statement

On Gap PDP, the current control experience does not sufficiently support checkout start rate for the first-time buyers segment. Based on prior quality-tactic learnings across the mid tier, we want to evaluate whether the proposed change moves the needle within the 28-day window.

1.2 Hypothesis

If we apply 'PDP UGC' on Gap PDP (Tablet Web) to first-time buyers, then we expect a lift of at least 0.48% on Checkout Start Rate because adding ugc photos to the pdp gallery improves perceived authenticity and atb.

1.3 Experimental Changes

  • Control: existing PDP experience on Gap Tablet Web.
  • Challenger: PDP UGC variant — adding ugc photos to the pdp gallery improves perceived authenticity and atb.

2. Test Results

2.1 Findings (TL;DR)

Across the 28-day test window, the challenger underperformed the control on the primary KPI (Checkout Start Rate) by 2.05% (statistically significant in the wrong direction). Net RPV moved by -2.15% and AOS by -0.14%.
