1. Test Plan

1.1 Problem Statement

On Old Navy Checkout, the current control experience does not sufficiently support checkout start rate for the cart abandoners segment. Based on prior value-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the 14-day window.

1.2 Hypothesis

If we apply 'Checkout EDD' on Old Navy Checkout (Mobile Web) to cart abandoners, then we expect a lift of at least 1.0% on Checkout Start Rate because surfacing the edd at checkout reduces last-minute drop-off.

1.3 Experimental Changes

  • Control: existing Checkout experience on Old Navy Mobile Web.
  • Challenger: Checkout EDD variant — surfacing the edd at checkout reduces last-minute drop-off.
