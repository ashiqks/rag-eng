1. Test Plan

1.1 Problem Statement

On Banana Republic Factory PDP, the current control experience does not sufficiently support checkout start rate for the loyalty members segment. Based on prior time savings-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the 21-day window.

1.2 Hypothesis

If we apply 'Sticky PDP ATB' on Banana Republic Factory PDP (Mobile Web) to loyalty members, then we expect a lift of at least 1.1% on Checkout Start Rate because a sticky atb on mobile pdp keeps the primary cta visible during long scrolls.

1.3 Experimental Changes

  • Control: existing PDP experience on Banana Republic Factory Mobile Web.
  • Challenger: Sticky PDP ATB variant — a sticky atb on mobile pdp keeps the primary cta visible during long scrolls.
