1. Test Plan

1.1 Problem Statement

On Old Navy PDP, the current control experience does not sufficiently support upt for the cart abandoners segment. Based on prior value-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the 60-day window.

1.2 Hypothesis

If we apply 'PDP points estimator' on Old Navy PDP (Mobile Web) to cart abandoners, then we expect a lift of at least 1.19% on UPT because showing 'you'll earn x points' on pdp reinforces loyalty value and lifts conversion among members.

1.3 Experimental Changes

  • Control: existing PDP experience on Old Navy Mobile Web.
  • Challenger: PDP points estimator variant — showing 'you'll earn x points' on pdp reinforces loyalty value and lifts conversion among members.
