1. Test Plan

1.1 Problem Statement

On Old Navy Search, the current control experience does not sufficiently support pdp -> shopping bag conversion for the non-loyalty segment. Based on prior time savings-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the 21-day window.

1.2 Hypothesis

If we apply 'Search recent chips' on Old Navy Search (Desktop) to non-loyalty, then we expect a lift of at least 0.25% on PDP -> Shopping Bag Conversion because recent search chips on the search overlay reduce typing.

1.3 Experimental Changes

  • Control: existing Search experience on Old Navy Desktop.
  • Challenger: Search recent chips variant — recent search chips on the search overlay reduce typing.
