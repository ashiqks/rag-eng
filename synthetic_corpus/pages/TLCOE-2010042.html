1. Test Plan

1.1 Problem Statement

On Gap Factory Checkout, the current control experience does not sufficiently support aur for the non-loyalty segment. Based on prior time savings-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the 90-day window.

1.2 Hypothesis

If we apply 'Guest checkout default' on Gap Factory Checkout (Tablet Web) to non-loyalty, then we expect a lift of at least 0.38% on AUR because defaulting to guest checkout (with optional signup at the end) lifts checkout completion.

1.3 Experimental Changes

  • Control: existing Checkout experience on Gap Factory Tablet Web.
  • Challenger: Guest checkout default variant — defaulting to guest checkout (with optional signup at the end) lifts checkout completion.
