1. Test Plan

1.1 Problem Statement

On Banana Republic PLP, the current control experience does not sufficiently support aur for the loyalty members segment. Based on prior quality-tactic learnings across the premium tier, we want to evaluate whether the proposed change moves the needle within the 90-day window.

1.2 Hypothesis

If we apply 'PLP affinity sort' on Banana Republic PLP (Android Push) to loyalty members, then we expect a lift of at least 0.39% on AUR because ml-ranked sort using prior browsing affinity should outperform default 'featured' sort.

1.3 Experimental Changes

  • Control: existing PLP experience on Banana Republic Android Push.
  • Challenger: PLP affinity sort variant — ml-ranked sort using prior browsing affinity should outperform default 'featured' sort.
