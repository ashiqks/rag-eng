1. Test Plan

1.1 Problem Statement

On Gap PDP, the current control experience does not sufficiently support checkout start rate for the non-loyalty segment. Based on prior quality-tactic learnings across the mid tier, we want to evaluate whether the proposed change moves the needle within the 120-day window.

1.2 Hypothesis

If we apply 'PDP AI fit' on Gap PDP (Tablet Web) to non-loyalty, then we expect a lift of at least 1.35% on Checkout Start Rate because an ai-driven fit recommender (à la zara) reduces size returns and lifts atb.

1.3 Experimental Changes

  • Control: existing PDP experience on Gap Tablet Web.
  • Challenger: PDP AI fit variant — an ai-driven fit recommender (à la zara) reduces size returns and lifts atb.
