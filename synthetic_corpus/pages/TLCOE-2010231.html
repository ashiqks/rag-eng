1. Test Plan

1.1 Problem Statement

On Gap Factory Checkout, the current control experience does not sufficiently support checkout start rate for the all customers segment. Based on prior value-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the -136-day window.

1.2 Hypothesis

If we apply 'Checkout EDD' on Gap Factory Checkout (Tablet Web) to all customers, then we expect a lift of at least 0.53% on Checkout Start Rate because surfacing the edd at checkout reduces last-minute drop-off.

1.3 Experimental Changes

  • Control: existing Checkout experience on Gap Factory Tablet Web.
  • Challenger: Checkout EDD variant — surfacing the edd at checkout reduces last-minute drop-off.
