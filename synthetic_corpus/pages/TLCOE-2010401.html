1. Test Plan

1.1 Problem Statement

On Athleta PLP, the current control experience does not sufficiently support checkout completion rate for the loyalty members segment. Based on prior quality-tactic learnings across the premium tier, we want to evaluate whether the proposed change moves the needle within the 75-day window.

1.2 Hypothesis

If we apply 'PLP affinity sort' on Athleta PLP (Tablet Web) to loyalty members, then we expect a lift of at least 0.32% on Checkout Completion Rate because ml-ranked sort using prior browsing affinity should outperform default 'featured' sort.

1.3 Experimental Changes

  • Control: existing PLP experience on Athleta Tablet Web.
  • Challenger: PLP affinity sort variant — ml-ranked sort using prior browsing affinity should outperform default 'featured' sort.
