1. Test Plan

1.1 Problem Statement

On Old Navy PDP, the current control experience does not sufficiently support checkout start rate for the cart abandoners segment. Based on prior urgency-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the 90-day window.

1.2 Hypothesis

If we apply 'PDP scarcity label' on Old Navy PDP (Tablet Web) to cart abandoners, then we expect a lift of at least 0.31% on Checkout Start Rate because a scarcity label on pdp triggers urgency and lifts atb rate.

1.3 Experimental Changes

  • Control: existing PDP experience on Old Navy Tablet Web.
  • Challenger: PDP scarcity label variant — a scarcity label on pdp triggers urgency and lifts atb rate.
