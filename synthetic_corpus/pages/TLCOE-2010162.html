1. Test Plan

1.1 Problem Statement

On Gap Factory Search, the current control experience does not sufficiently support upt for the new visitors segment. Based on prior time savings-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the 8-day window.

1.2 Hypothesis

If we apply 'Search recent chips' on Gap Factory Search (Mobile Web) to new visitors, then we expect a lift of at least 0.37% on UPT because recent search chips on the search overlay reduce typing.

1.3 Experimental Changes

  • Control: existing Search experience on Gap Factory Mobile Web.
  • Challenger: Search recent chips variant — recent search chips on the search overlay reduce typing.
