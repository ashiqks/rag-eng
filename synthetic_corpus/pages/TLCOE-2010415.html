1. Test Plan

1.1 Problem Statement

On Athleta PDP, the current control experience does not sufficiently support checkout completion rate for the new visitors segment. Based on prior value-tactic learnings across the premium tier, we want to evaluate whether the proposed change moves the needle within the 42-day window.

1.2 Hypothesis

If we apply 'PDP points estimator' on Athleta PDP (Mobile Web) to new visitors, then we expect a lift of at least 0.26% on Checkout Completion Rate because showing 'you'll earn x points' on pdp reinforces loyalty value and lifts conversion among members.

1.3 Experimental Changes

  • Control: existing PDP experience on Athleta Mobile Web.
  • Challenger: PDP points estimator variant — showing 'you'll earn x points' on pdp reinforces loyalty value and lifts conversion among members.
