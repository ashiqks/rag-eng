1. Test Plan

1.1 Problem Statement

On Old Navy Checkout, the current control experience does not sufficiently support checkout start rate for the non-loyalty segment. Based on prior time savings-tactic learnings across the discount tier, we want to evaluate whether the proposed change moves the needle within the 53-day window.

1.2 Hypothesis

If we apply 'Wallet pay promotion' on Old Navy Checkout (iOS Native) to non-loyalty, then we expect a lift of at least 1.37% on Checkout Start Rate because promoting wallet pay above the billing form shortens checkout for mobile users.

1.3 Experimental Changes

  • Challenger: Wallet pay promotion variant — promoting wallet pay above the billing form shortens checkout for mobile users.
